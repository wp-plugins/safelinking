This modification integrates safelinking.net with your website

== Description ==
This modification, fully compatible with any WordPress 3 version, will add a block (textarea and button) to your user and wp-admin posting page where you and your users will be able to generate safelinking.net protected links by a single click, without going outside your site.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're done!
